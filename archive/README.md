# Archive Directory

このディレクトリには、開発過程で作成されたバックアップファイルが保存されています。

## 分類
- database/: データベース関連のバックアップ
- config/: 設定ファイルのバックアップ  
- templates/: テンプレートファイルのバックアップ
- static/: 静的ファイルのバックアップ
- logs/: ログファイルのバックアップ
- other/: その他のバックアップファイル

## 注意事項
- これらのファイルは本番環境には不要です
- .dockerignoreで除外されています
- 必要に応じて参照・復元に使用してください
